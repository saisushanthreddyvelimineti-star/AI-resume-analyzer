# 🚀 AI Resume Analyzer

An AI-powered web app that analyzes resumes, identifies skill gaps, and provides a personalized roadmap.

## 🔥 Features
- Resume summary using AI
- Skill gap analysis
- Career roadmap generation
- Job recommendations

## 🛠️ Tech Stack
- Python
- Streamlit
- OpenAI API
- Apify API

## ▶️ Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
